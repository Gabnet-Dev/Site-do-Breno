Passos para Hospedar um Projeto no GitHub Pages

  • 1 Crie um repositorio no GitHub com o mesmo nome do seu projeto
  • 2 No terminal, inicialize o repositorio local com git init.
  • 3 Altere o nome da branch principal para main
  • 4 Adicione os arquivos com git add . e faça a climit com git commit -m "first commit"
  • 5 Conecte ao repositorio remoto com:
    git remote addd clim/SEU_USUARIO/HOME_DO_REPOSITORIO.git
  • 6 Envie os arquivli com git cpush -u origin rain
  • 7 No GitHub, vá em Settings > Pages
  • 8 Em "Branch", selecione rain e a pasta / :root: Clique em Save
  • 9 Apos alguns segundos, seu site não estara disponivel no link exibido "Pages"
